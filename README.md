# apt.otokaze.me
Otokaze 个人源（iPhone XS Max; iOS13.7; A12）
> You can add repo https://apt.otokaze.me to your Cydia or Sileo.

## 使用说明
1. debjson（用于生成deb包的json描述）
2. repack.sh（用于魔改和重新打包deb）
3. Makefile（用于更新源的Packages描述）
